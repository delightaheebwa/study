# PBL Mission: AI Engineering from Scratch — Project-Based Learning

## Why I'm Doing This
I want to master AI engineering by building real production systems, not by watching lectures or reading theory in isolation. I learn best when I have a concrete project goal and pull in the foundational concepts as I need them.

## The Goal
By the end of this curriculum, I will have built 19 production-grade AI systems:
- RAG systems (codebase RAG + production regulated RAG)
- Agent systems (coding agent, DevOps agent, migration agent, GitHub PR agent, multi-agent SWE team)
- Multimodal systems (document QA, video understanding, voice assistant, personal tutor)
- Training infrastructure (fine-tuning pipeline, speculative decoding server)
- Safety systems (constitutional safety harness, red-team range)
- Classical ML pipeline (customer churn prediction with MLOps)
- RL implementation ladder + RLHF
- Research agent (autonomous ML research)
- Observability dashboard (monitoring)

## My Commitment
I will build every project from first principles, understand the math behind each algorithm, and push every lesson to GitHub for backup and portfolio purposes.
